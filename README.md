# Проект «Оно тебе надо»

Одностраничный сайт-аукцион, выполненный в рамках обучения на курсе веб-разработки.  
Проект реализован по макету и сверстан с использованием методологии БЭМ.

## Описание

Сайт представляет собой рекламную страницу аукциона с лотами, описанием и контактной информацией.  
Основная задача проекта — отработка навыков семантической вёрстки, организации файловой структуры и использования Flexbox.

## Технологии

- HTML5
- CSS3
- Flexbox
- Методология БЭМ
- Normalize.css
